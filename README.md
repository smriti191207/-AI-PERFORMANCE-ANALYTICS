# AI-Powered Employee Performance Analytics

## Overview

AI-Powered Employee Performance Analytics is an HR analytics project that combines traditional employee performance evaluation with Natural Language Processing (NLP)-based sentiment analysis.

The system analyzes employee and manager feedback comments, performance scores, ratings, and evaluation gaps to generate actionable workforce insights that support data-driven decision-making.

---

## Problem Statement

Performance reviews often contain valuable information hidden within textual feedback. Traditional HR reporting focuses mainly on scores and ratings, making it difficult to understand employee sentiment and identify performance trends.

This project uses AI and NLP techniques to extract meaningful insights from employee and manager review comments while combining them with performance evaluation data.

## Key Analytics Performed

### Sentiment Analysis

Classifies employee and manager comments as:

* Positive
* Neutral
* Negative

### Rating Analysis

Converts textual performance ratings into numeric values for analysis.

### Gap Analysis

Measures differences between:

* Employee and Manager Scores
* Employee and Manager Ratings
* Employee and Manager Sentiment

### Correlation Analysis

Identifies relationships between:

* Performance Scores
* Ratings
* Sentiment Scores
* Promotion Recommendations

---

## Sample Insights

* High-performing employees generally receive positive sentiment scores.
* Employee and manager perceptions can differ significantly in certain cases.
* Promotion recommendations show strong alignment with overall performance ratings.
* Sentiment analysis provides additional context beyond numerical scores.
